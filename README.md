# Abominable Intelligence
Small Discord bot made with [Hikari-Lightbulb](https://github.com/tandemdude/hikari-lightbulb)

## Commands
| Main Command                         | Desc                                                                         |
| ------------------------------------ | ---------------------------------------------------------------------------- |
| /dice [dice] [sides]                 | Dice roller                                                                  |
| /gw2                                 | Guilds Wars 2 tools for thing like guild members nickname lookup             |
| /ai                                  | Ollama integration for AI prompts                                            |
| /git                                 | git commands to update the bot or change its branch                          |
| /settings                            | bot's settings                                                               |
